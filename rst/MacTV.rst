MacTV is a program for [[macOS]] computers that is based on
[[QuickTime]] software. It provides support for more features like Cisco
IP/TV, [[MBONE]] and [[SAP]]. Like QuickTime, it supports [[MPEG]],
[[H.261]] and [[QuickTime Video]]
