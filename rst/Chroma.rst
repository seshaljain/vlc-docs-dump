Greek for color, the part of a video file or signal that encodes the color portion.

List of Video Chromas
---------------------

-  I420

-  I411

-  I422

-  YUYV

-  UYVY

-  RV24 (basically this is RGB24, 3 bytes, one for R, G, B)

-  RV32

-  I42N

-  I41N

-  GRAW

See also `YUV <YUV>`__.
