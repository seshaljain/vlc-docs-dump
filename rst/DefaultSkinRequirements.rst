General requirements
--------------------

-  Be sobre, so as to fit in all environments
-  No real need for a big "VLC" or "VideoLAN" text/logo
-  Original design, not copied from something existing

Mandatory elements
------------------

-  Main window

   -  Play/Pause
   -  Stop
   -  Next
   -  Previous
   -  Slider
   -  Volume
   -  Time display
   -  DVD buttons (do we want them to be shown all the time ? -- of course not)
   -  Playlist button

-  Playlist

   -  Playlist (use )
   -  Add button

-  Video window

Optional elements
-----------------

-  Equalizer
-  Playlist

   -  Random, loop buttons
   -  Or : submenu

-  Video window

   -  Fullscreen button

-  Miniature mode
