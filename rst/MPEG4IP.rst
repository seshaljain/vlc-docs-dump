'''MPEG4IP''' is an open source [[MPEG]], [[H.261]], [[AAC]] and [[AMR]]
library. It can create MPEG-4 Files.

MPEG4IP is a software [[library]], rather than a program, but some
programs may use MPEG4IP when they run.
